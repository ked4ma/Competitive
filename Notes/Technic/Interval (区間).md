# 対称性のある区間を包含する区間の数え上げ
いくつかの区間が与えられたとき、
- x
- N - x
両方を含むものの数を知りたいとする
![[Screenshot from 2026-04-12 19-06-26.png]]
(この画像の区間2みたいなのの総数が知りたい)
そういうとき、区間(L..R)の扱いを下図の紫のようにして扱う
$$
\begin{gather}
a = max(L, N - R) \\
(L, R) \rightarrow (a, N - a)
\end{gather}
$$
![[Screenshot from 2026-04-12 19-09-42.png]]
xとN - xがどちらも終端か
探したい区間は x, N - xをどちらも自身の区間に含むものだが、これはx, N - xのどちらも終端から等しくxだけ離れていることを利用する。
このようにして扱った区間をimos法などで数え上げておけば、
「x, N - xをどちらも含むもの」→ 「xを含むもの」という探し方に落とし込める
